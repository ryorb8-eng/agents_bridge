# Geometry_Engine — Curated Knowledge Bank (Z-sourced)

Knowledge TERKURASI yang diverifikasi dari balasan **Z Web** untuk instance
Geometry Engine. Dipisah dari `answers_import/bank_knowledges/` (ChatGPT-sourced) dan
`claude_answers_import/bank_knowledges/` (Claude-sourced) agar ketiga sumber remote-AI
tidak tercampur.

- Sumber: `z_answers_import/temp_answers.md` → verifikasi
  (`temp_knowledges/VERIFY-Q<n>.md`) → curation ke sini.
- HANYA orchestrator (curate phase `/webchain-z`) yang menulis bank ini.
- Format tiap claim: `**Claim** — <teks>. *(Q<n>, conf <x>%)*`.
- REJECT → tidak masuk bank, diarsip di `temp_knowledges/`.

> Belum ada knowledge (Z chain belum dijalankan).
