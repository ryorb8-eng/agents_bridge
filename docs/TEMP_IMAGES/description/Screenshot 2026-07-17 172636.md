# 2. Executive Summary
Gambar merupakan screenshot antarmuka aplikasi editor bernama **"5. SKELETONE EDITOR"** yang digunakan untuk mendesain skeleton (kerangka) frame kacamata berbasis anchor dan kurva.
Bagian utama menampilkan preview bentuk frame lengkap dengan titik-titik anchor yang dapat diedit secara langsung.
Toolbar di bagian atas menyediakan operasi edit seperti Select, Anchor, Add/Delete Anchor, Clone, Mirror, Rotate, dan Curve.
Panel kanan menampilkan metadata objek frame beserta informasi anchor yang sedang dipilih.
Panel bawah menyediakan kontrol transformasi, border, material, warna, opacity, serta aksi penyimpanan dan ekspor SVG.
Mode editor sedang berada pada status **EDIT**.
UI ini tampak dirancang sebagai editor internal untuk membangun model frame kacamata berbasis vektor.
